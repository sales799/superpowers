# Brain — Project Intelligence Files

Central repository of CLAUDE.md project intelligence files used across all Talpro/HCI ventures.

## Structure

```
brain/
  global/CLAUDE.md          # Universal instructions (all projects)
  projects/
    _shared/CLAUDE.md       # Shared context across projects
    929digital/CLAUDE.md    # 929 Digital website
    catalyzer-labs/CLAUDE.md # Catalyzer Labs
    competitorx/CLAUDE.md   # CompetitorX
    elovia/CLAUDE.md        # Elovia
    hci-intel/CLAUDE.md     # HCI Intel analytics
    hcitalks/CLAUDE.md      # HCITalks (MCP summary)
    hcitalks-full/CLAUDE.md # HCITalks (full project intelligence)
    jharokha/CLAUDE.md      # Jharokha smart doorbell
    leadhunter/CLAUDE.md    # LeadHunter sales automation
    talpro-india/CLAUDE.md  # Talpro India staffing
    tara-cco/CLAUDE.md      # TARA CCO agent
```

## Usage

These files provide project context to Claude Code, enabling it to understand each project's architecture, conventions, and constraints. They are consumed by the talpro-mcp server and local Claude Code sessions.
